# 📊 Data Analysis Workbench

This repository is a collection of my hands-on practice while learning data analysis using Python.
I created this to move beyond just “knowing syntax” and actually understand:

* how data is cleaned
* how analysis flows step by step
* how different libraries work together in real scenarios

---

## 🚀 Why I Built This

I started building this repo as a **personal reference + practice space** where I:
* experiment with datasets
* revisit concepts
* and gradually build confidence in real workflows

---

## 📁 How This Repo Is Organized
I tried to organize things based on how I learn:

* `numpy/` → understanding arrays and numerical operations
* `pandas/` → working with real datasets, cleaning and transforming
* `matplotlib/` → basic plotting and visual understanding
* `seaborn/` → improving visualizations and exploring patterns
* `mixed/` → combining everything into actual workflows

Each folder contains:

* All the files needed to run the notebook along as I might have used some resources repetitively
* My comments which helped me understand key details about what I learn so I can revisit it again and again , maybe so can you
